A fun, feature-packed Discord entertainment bot.

## 🔗 Bot Page
You can find Clanker on top.gg here:  
https://top.gg/bot/1482397035909873865  

If you enjoy the bot, consider leaving a review or vote - it helps a lot.

## ⚠️ Self Hosting  
Clanker is primarily designed to be used as a hosted bot.  

The source code is available so the community can:
- report bugs  
- suggest improvements  
- contribute features  

Self-hosting is possible, but not officially supported, and you may run into issues setting everything up.
