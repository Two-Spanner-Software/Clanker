## ⚠️ Self Hosting  
Clanker is primarily designed to be used as a hosted bot.  

The source code is available so the community can:
- report bugs  
- suggest improvements  
- contribute features  

Self-hosting is possible, but not officially supported, and you may run into issues setting everything up.
